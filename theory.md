# Probability & Statistics for Machine Learning

## Topics Covered
- Probability Basics
- Conditional Probability
- Bayes Theorem
- Mean, Variance, Standard Deviation
- Distributions (Normal, Bernoulli, Binomial)

## Key Formulas

P(A) = favorable outcomes / total outcomes

P(A|B) = P(A ∩ B) / P(B)

Bayes Theorem:
P(A|B) = (P(B|A) * P(A)) / P(B)

Mean:
μ = (1/n) * Σxi

Variance:
σ² = (1/n) * Σ(xi - μ)²
